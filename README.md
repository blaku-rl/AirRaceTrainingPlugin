# AirRaceTrainingPlugin
A plugin that allows the player to teleport to each of the check points in Panics Air Race map.

The Air Race Training Plugin is intended to be used with Panic's Air Race map. The map can be found in the steam workshop. The plugin allows you to teleport to any of the checkpoints on the map.

The bakkesmod commands for the teleport follow the format of airracetraining\_panicsairrace\_{level}check, where level is First, Second, Third... As Panic makes new maps, I intend to update this plugin to work with them. I recommend binding the check point you want to work to a dpad input on controller and a number on kbm.

**Important** you must reset to the beginning of the map before teleporting to any check point.
